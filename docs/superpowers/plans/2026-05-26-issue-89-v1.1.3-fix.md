## Task 3: SettingsView 末尾に `.safeAreaInset` で spacer 追加 (#4)

**Files:**
- Modify: `app/OtetsudaiCoin/Presentation/Views/SettingsView.swift` (List modifier 群、おそらく line 200 以降の closing 周辺)

**背景:** `NavigationStack + List` 構造で `.safeAreaInset()` / `.contentMargins()` 未使用、`BannerAdView` 配置なし → tab bar 被りが唯一の要因。`.safeAreaInset(edge: .bottom)` で 16pt の透明 spacer を追加し、最後の Section が tab bar に重ならないようにする。

- [ ] **Step 1: SettingsView.swift の現状確認**

```bash
sed -n '60,70p;195,210p' /Users/shinya/workspace/claude/OtetsudaiCoin/app/OtetsudaiCoin/Presentation/Views/SettingsView.swift
```

Expected: line 62 付近に `NavigationStack { List { ... } }` 構造、 line 200 付近に `Section("アプリ情報")` の closing。

- [ ] **Step 2: List に `.safeAreaInset(edge: .bottom)` modifier を追加**

List の末尾 (NavigationStack の closing `}` の直前) に以下の modifier を追加。

```swift
.safeAreaInset(edge: .bottom) {
    Color.clear.frame(height: 16)
}
```

例 (調整後の構造イメージ):
```swift
NavigationStack {
    List {
        // ... 既存の Section 群 ...
        Section("アプリ情報") {
            HStack {
                Image(systemName: "info.circle")
                    .foregroundColor(.blue)
                Text("バージョン")
                Spacer()
                Text(appVersionText)
                    .foregroundColor(.secondary)
            }
        }
    }
    .safeAreaInset(edge: .bottom) {  // ← 追加
        Color.clear.frame(height: 16)
    }
    // ... 既存の navigationTitle 等 ...
}
```
