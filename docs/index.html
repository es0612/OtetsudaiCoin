おてつだいコイン

子供のお手伝いを楽しく記録して、コインを獲得してお小遣い管理ができる iOS アプリ。

主な機能

お手伝いを記録 お子様とタスクを選ぶだけ。コイン獲得アニメーション・効果音・触覚フィードバックでお子様のモチベーションを高めます。
お小遣い管理 獲得コインから月別お小遣いを自動計算。月末支払いの通知やリマインダーで、保護者の管理負担を軽減します。
履歴を振り返り 今週・今月・過去 3 か月・全期間のフィルタで履歴を確認。月別履歴では未払いの状況も一目でわかります。
家族で使える 複数のお子様を登録して、それぞれにテーマカラーを設定。個性を尊重しながら家族みんなで使えます。

動作環境

iOS 18.5 以降の iPhone / iPad に対応。
